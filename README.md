## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A text editor or IDE (VS Code, Sublime Text, etc.)
- Git installed on your machine

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mnxtr/mnxtr.github.io.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd mnxtr.github.io
   ```

3. **Open the website:**
   Simply open `index.html` in your web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # On Windows
   start index.html
   ```

   Or use a local development server (recommended):
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

4. **View the site:**
   Open your browser and navigate to `http://localhost:8000`

## 🌐 Deployment

This portfolio is automatically deployed to GitHub Pages. Any changes pushed to the main branch will be reflected on the live site.

### Manual Deployment

1. Ensure all changes are committed:
   ```bash
   git add .
   git commit -m "Your commit message"
   ```

2. Push to GitHub:
   ```bash
   git push origin main
   ```

3. The site will be automatically deployed to `https://mnxtr.github.io`

### Custom Domain (Optional)

To use a custom domain:
1. Add a `CNAME` file to the repository root with your domain
2. Configure DNS settings with your domain provider
3. Enable HTTPS in GitHub Pages settings
