## 🛠️ Customization Guide

### Change Colors
Edit `css/style.css` line 11-16:
```css
:root {
    --primary-color: #006A4E;
    --secondary-color: #F42A41;
    --accent-color: #FFB800;
}
```

### Add Gradio Demo
Edit `index.html` line 463, replace placeholder with:
```html
<iframe src="YOUR-GRADIO-URL" width="100%" height="600px"></iframe>
```

### Add Google Analytics
Add before `</head>` in `index.html`:
```html
<script async src="https://www.googletagmanager.com/gtag/js?id=GA-ID"></script>
```
